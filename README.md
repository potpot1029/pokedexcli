# pokedexcli

my implementation for "Build a Pokedex" project on [boot.dev](https://www.boot.dev/).

## how to use

start the program with:

```shell
go run .
```

in the program, check what you can do with `help command`:

```shell
Pokedex > help
Welcome to the Pokedex!
Usage:

help: Displays a help message
exit: Exit the Pokedex
map: Displays the names of next 20 location areas in the Pokemon world
mapb: Displays the names of previous 20 location areas in the Pokemon world
explore <location-area>: See a list of all Pokemon in <location-area>
catch <pokemon-name>: Catch <pokemon-name> with a Pokeball
inspect <pokemon-name>: Inspect information about caught <pokemon-name>
pokedex: List all caught Pokemon(s)
```
